# Causal Inference for The Brave and True: 한국어 강의노트

숫자는 그럴듯해 보일 때가 많습니다.

비싼 마우스를 쓰는 사람의 승률이 높다면, 마우스가 승률을 올린 걸까요? 아니면 원래 게임을 잘하는 사람이 비싼 마우스를 산 걸까요?

이 노트는 이런 질문을 하나씩 따라가며 읽는 한국어 강의노트입니다.

원본 `Causal Inference for The Brave and True`를 참고하지만, 문장별 번역은 아닙니다. 처음 읽는 사람도 한 장씩 따라갈 수 있도록 예시와 설명 순서를 새로 잡았습니다.

각 장은 익숙한 상황에서 시작합니다. 먼저 왜 쉬운 비교가 위험한지 보고, 그다음에 필요한 이름, 작은 계산, 파이썬 코드를 붙입니다.
