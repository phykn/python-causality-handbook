# Causal Inference for The Brave and True: 한국어 강의노트

인과추론은 어떤 선택이나 행동이 결과를 실제로 바꾸었는지 판단하는 방법입니다.

데이터를 보면 두 일이 함께 보일 때가 많습니다. 하지만 함께 보인다는 것만으로는 한쪽이 다른 쪽의 원인이라고 말하기 어렵습니다.

그래서 인과추론은 먼저 비교를 묻습니다. 어떤 사람들과 어떤 사람들을 비교해야 하는지, 그 비교가 공정하려면 무엇이 비슷해야 하는지, 그래도 남는 의심은 무엇인지 차례대로 확인합니다.

이 저장소는 `Causal Inference for The Brave and True`를 참고해 작성하는 한국어 강의노트입니다.
